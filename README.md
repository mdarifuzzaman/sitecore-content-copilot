# Sitecore Content Copilot

Sitecore Content Copilot is a VS Code extension that helps Sitecore XM Cloud and Headless developers inspect content, generate code, and work with GraphQL faster.

## Features

### Sitecore Explorer
Browse Sitecore content directly from VS Code.

### GraphQL Explorer
Run queries against your Sitecore GraphQL endpoint.

### Generate JSS Models
Generate strongly typed JSS field models from Sitecore items.

### Generate React Components
Create starter Next.js/JSS components from Sitecore fields.

### Copy GraphQL Query
Generate a GraphQL query for the selected item.

## Requirements

- Sitecore XM Cloud / GraphQL endpoint
- API Key with access
